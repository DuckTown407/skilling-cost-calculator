# Skilling Cost Calculator

This plugin is inteded to calculate the cost/profit/loss for skilling methods. You can pretty easily figure out the initial cost of a skilling method with the skill calculator plugin, but most times you can sell the items back and recoup some of the cost. This provides a convenient way to check skilling methods against current GE prices. 

## Current scope

- Much like the skill calculator this plugin can read your current skill level/XP and allows manual input

- Automatically attemps to read your current XP when switching skills
- You can choose what level/XP you'd like to get to
- Provides a list of available methods than be sorted by least action or most profit (least lost in most cases)
  <img width="246" height="394" alt="SCR-20260618-mviy" src="https://github.com/user-attachments/assets/a8205409-7903-4bf9-9ab5-b3e690247e8d" />

- Method list is scrollable
- Method list highlights if you have the required skill level for the method

- Optional toggle to hide methods you don't mee the requirements for
- Uses live GE proficing to calculate profit/loss
- Toggle to include/exclude GE tax
- Click the button to run the calculation

- Optionally add the cost of extra supplies (like thread for d'hide)
- Able to look at smithing methods by buying bars or making bars
